# Smart Plant Monitoring System (ESP32)

## Overview
An IoT system to monitor plant conditions using sensors and cloud integration.

## Features
- Temperature & humidity monitoring
- Soil moisture detection
- Cloud visualization

## Components
- ESP32
- DHT11
- Soil Moisture Sensor
- ThingSpeak, Blynk

## Working
Sensors collect data → ESP32 processes → sends to cloud → user monitors remotely.

## Applications
- Smart agriculture
- Remote monitoring
